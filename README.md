# 지역별 사회경제적 취약성과 범죄율의 관계 분석 (2023)

## 1. 프로젝트 개요

본 프로젝트는 2023년 공공데이터를 활용하여  
지역별 **기초생활보장 수급자 비율**과 **범죄율(인구 10만명당)** 간의 관계를 분석한 프로젝트입니다.

단순 통계 분석에 그치지 않고,  
서로 다른 출처의 공공 데이터를 **지역·연도 기준 관계형 구조로 통합 설계**하는 과정에 중점을 두었습니다.

---

## 2. 사용 데이터

- 경찰청 – 지역별 범죄 발생 통계 (2023)
- 보건복지부 – 기초생활보장 수급자 수 (2023)
- 행정안전부 – 주민등록 인구 통계 (2023)

원본 데이터는 공공데이터포털 공개 자료입니다.

---

## 3. 데이터 통합 및 모델 설계

### 3.1 데이터 정규화 과정

- 지역명 표기 방식 통일
- 행정코드 기준 매핑
- 연도 기준 통일
- 단위(비율, 인구 기준) 표준화
- 중복 레코드 제거 및 결측치 처리 기준 정의

---

### 3.2 관계형 데이터 모델 설계

서로 다른 데이터 소스를 통합하기 위해  
지역과 연도를 기준으로 다음과 같이 모델을 설계하였습니다.

#### dim_region
- region_id (PK)
- region_name
- region_code
- administrative_level

#### dim_year
- year_id (PK)
- year

#### fact_welfare
- region_id (FK)
- year_id (FK)
- basic_livelihood_cnt
- welfare_ratio

#### fact_crime
- region_id (FK)
- year_id (FK)
- total_crime_cnt
- crime_rate

모든 데이터는 `(region_id, year_id)` 복합 키를 기준으로 연결되도록 설계하였습니다.

---

### 3.3 인덱스 및 조회 패턴 고려

- `(region_id, year_id)` 복합 인덱스 설계
- 연도 단위 집계 최적화 구조 적용
- 지역 기준 통합 분석을 위한 키 일관성 유지

---

## 4. 분석 방법

1. 지역 기준 데이터 병합  
2. 파생 변수 생성  
   - 수급자비율(%) = (수급자수 ÷ 총인구수) × 100  
   - 범죄율(10만명당) = (범죄건수 ÷ 총인구수) × 100,000  
3. 피어슨 상관 분석  
4. 이상치(세종, 제주) 제거 후 재분석  
5. 단변량 OLS 회귀 분석  

---

## 5. 주요 결과

- 전체 지역 기준 상관계수: **0.52**
- 이상치 제거 후 상관계수: **0.466**
- 회귀 분석 설명력(R²): **0.217**
- 수급자 비율이 1% 증가할 때 범죄율은 약 **128건(10만명당)** 증가하는 경향

두 변수는 양의 상관관계를 보였으나,  
상관관계는 인과관계를 의미하지 않습니다.
